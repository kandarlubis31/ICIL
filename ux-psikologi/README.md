# 🧠 Faculty of UX Psychology — Overview

> **7 Courses** | Level: Intermediate → Advanced | Domain: Behavioral Design & Cognitive Science

This faculty unpacks how the human brain works when interacting with digital products. From cognitive biases, fundamental UX laws, Gestalt principles, to ethical persuasion techniques and gamification. The key to building products that "understand humans."
