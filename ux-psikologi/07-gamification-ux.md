# 🧠 07 — Gamification in UX

> 🔴 Advanced | Prereq: 05 | ~4 min

Gamification = game elements in non-game contexts → increase engagement, motivation, retention. NOT making everything a game — borrowing mechanics that make games work.

---

## 7.1 Self-Determination Theory (SDT)

| Human Need | Gamification Mechanic | Example |
|------------|----------------------|---------|
| **Autonomy** (Control) | Choices, customization | Avatar, profile themes |
| **Competence** (Mastery) | Levels, badges, progress | Skill trees, XP, achievements |
| **Relatedness** (Connection) | Social interaction | Leaderboards, team quests, share |

---

## 7.2 Octalysis — 8 Core Drives (Yu-kai Chou)

| Drive | Mechanic |
|-------|----------|
| Epic Meaning | Feel part of something bigger |
| Accomplishment | Progress, badges, level-ups |
| Empowerment | Creativity, feedback loops |
| Ownership | Collection, customization, virtual goods |
| Social Influence | Leaderboards, friends, competition |
| Scarcity | Exclusive access, limited items |
| Unpredictability | Random rewards, mystery boxes |
| Avoidance | Fear of losing progress/streaks |

---

## 7.3 Mechanics & Spectrum

| Mechanic | How It Works | Risk |
|----------|-------------|------|
| **Points** | Accumulation = progress | Shallow if not tied to value |
| **Badges** | Visual achievement = status | Users chase badges, not outcomes |
| **Streaks** | Fear of breaking daily chain | Addiction (Duolingo, Snapchat) |
| **Leaderboards** | Competition + comparison | Demotivates bottom 90% |
| **Variable Rewards** | Unpredictable = most addictive | Gambling-like (loot boxes) |

| Level | Example | Risk |
|-------|---------|------|
| Light | Onboarding progress bar | Minimal |
| Medium | Badges, streaks, levels | Moderate |
| Heavy | Leaderboards, quests, rewards | Engagement manipulation |
| **Danger** | Loot boxes, gambling-like | Addiction, **unethical** |

---

## 7.4 Pitfalls

| Pitfall | Solution |
|---------|----------|
| Shallow engagement (chase points, not value) | Points must correlate with meaningful action |
| Overjustification (external rewards kill internal motivation) | Don't gamify actions users already naturally do |
| Addiction (variable rewards exploit dopamine) | Ethical boundaries; always provide opt-out |

---

## ⚡ Action Checklist
- [ ] Map gamification to user's intrinsic needs (SDT: autonomy, competence, relatedness)
- [ ] Ensure points/badges reward meaningful actions — not just clicks
- [ ] Avoid overjustification: don't gamify already-enjoyed tasks
- [ ] Provide clear opt-out for ALL gamification mechanics
- [ ] No loot boxes or gambling-like mechanics — ethical boundary
