# 🧠 05 — Persuasion & Behavioral Design

> 🔴 Advanced | Prereq: 02 | ~4 min

How to drive user behavior ethically — without manipulation. Covers Cialdini, Fogg, Hook Model, Nudge Theory.

---

## 5.1 Cialdini's 7 Principles

| # | Principle | Core Idea | UX Tactic |
|---|-----------|-----------|-----------|
| 1 | **Reciprocity** | Give first → user feels obligated to return | Free trial, free tool, free value before asking |
| 2 | **Commitment** | Small yes → bigger yes later | Email signup → profile → upgrade |
| 3 | **Social Proof** | Follow what others do | "10K+ users", testimonials, real-time activity |
| 4 | **Authority** | Comply with experts | Trust badges, certifications, endorsements |
| 5 | **Liking** | Influenced by those we like | Friendly tone, real photos (not stock), relatable brand |
| 6 | **Scarcity** | Rare = valuable | "Only 3 left", "Sale ends in 2h" (must be REAL) |
| 7 | **Unity** | Shared identity = trust | "For developers, by developers", community access |

## 5.2 Fogg Behavior Model (B=MAP)

```
Behavior = Motivation × Ability × Prompt
If ANY = 0 → behavior doesn't happen.
```

| Factor | What It Means | UX Priority |
|--------|--------------|-------------|
| **Motivation** | Pleasure/Pain, Hope/Fear, Social Accept/Reject | Increase if low |
| **Ability** | Time, money, effort, brain cycles | **MOST IMPORTANT** — make it EASIER > more motivation |
| **Prompt** | Spark (low motiv.), Facilitator (low ability), Signal (both high) | Match type to gap |

> **Rule**: Increasing ability beats increasing motivation every time. Simplify, don't persuade harder.

---

## 5.3 The Hook Model (Nir Eyal)

```
TRIGGER → ACTION → VARIABLE REWARD → INVESTMENT → (back to TRIGGER)
```

| Phase | Description | Example |
|-------|------------|---------|
| **Trigger** | External (notification) or Internal (boredom, FOMO) | Push notification |
| **Action** | Simplest behavior for reward | Scroll, pull-to-refresh, like, tap |
| **Reward** | Tribe (likes), Hunt (feed), Self (achievement) | Variable = unpredictable = addictive |
| **Investment** | User puts in value → more likely to return | Followers, content, settings, streaks |

> ⚠️ Variable rewards = slot machine effect. Use ethically.

---

## 5.4 Nudge Theory

Gentle pushes that guide without limiting freedom:

| ✅ Ethical Nudge | ❌ Not a Nudge |
|-----------------|---------------|
| Default opt-out (not opt-in) | Hide cancel button |
| "Most popular" label on pricing | Pre-checked opt-in |
| Smaller portions as default | Hidden costs |

---

## ⚡ Action Checklist
- [ ] Offer free value BEFORE asking for commitment (Reciprocity)
- [ ] Simplify tasks rather than trying to increase motivation (Fogg)
- [ ] Use variable rewards sparingly — never exploit dopamine loops
- [ ] Verify: can users easily opt out / cancel / skip? (Nudge boundary)
- [ ] Every persuasion tactic must pass the "Mom Test": would she say it's fair?
