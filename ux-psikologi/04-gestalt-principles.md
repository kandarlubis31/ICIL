# 🧠 04 — Gestalt Principles in UX

> 🟡 Intermediate | Prereq: 01 | ~3 min

Brain organizes visual elements into **meaningful wholes**, not separate parts. "The whole is greater than the sum of its parts."

---

## 4.1 The 7 Principles

| # | Principle | Rule | UI Application | ❌ Violation |
|---|-----------|------|---------------|-------------|
| 1 | **Proximity** | Close = related | Label near its input; cards group info | Label far from input → confusion |
| 2 | **Similarity** | Same look = related | All primary buttons same color; all links underlined | Mix red & blue links → what's the difference? |
| 3 | **Continuity** | Eye follows smooth paths | Timeline dots connected; aligned elements flow | Unnecessary zigzag layouts |
| 4 | **Closure** | Brain completes incomplete shapes | Dashed icons, loading skeletons, carousel partial peek | Too abstract icon → unrecognized |
| 5 | **Figure-Ground** | Brain separates foreground/background | Modal + blurred bg; card on surface; active nav item | Ambiguous → user doesn't know what's interactive |
| 6 | **Common Fate** | Moving together = one group | Dropdown slides together; carousel; accordion expand | — |
| 7 | **Pragnanz** | Simplify to simplest form | Simple icons > detailed; symmetrical > asymmetrical | Overly detailed icons → slow processing |

---

## 4.2 Quick Application Map

| UI Element | Gestalt Applied |
|------------|----------------|
| Form layout | Proximity (label near input) |
| Button design | Similarity (all CTAs same color) |
| Modal dialog | Figure-Ground (blurred backdrop) |
| Loading state | Closure (skeleton screen) |
| Dropdown menu | Common Fate (items move together) |
| Icon system | Pragnanz (simple, consistent) |

---

## ⚡ Action Checklist
- [ ] Group related elements with proximity (reduce whitespace within groups, increase between)
- [ ] Use ONE color for all primary CTAs (Similarity)
- [ ] Add backdrop blur/scrim for modals (Figure-Ground)
- [ ] Keep icons simple and consistent in style (Pragnanz)
- [ ] Animate menus/accordions as a group (Common Fate)
