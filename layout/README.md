# 📐 Faculty of Layout & Grid — Overview

> **7 Courses** | Level: Beginner → Advanced | Domain: Layout Systems & Composition

This faculty covers the entire spectrum of digital layout: from basic grid theory, CSS Grid (basics to subgrid), complete Flexbox, responsive breakpoints, visual composition, to golden ratio and spacing systems. Designed so AI agents can create precise, responsive, and aesthetic layouts.
