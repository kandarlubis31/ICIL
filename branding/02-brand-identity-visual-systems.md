# 🏷️ 02 — Brand Identity & Visual Systems

> 🟢 Beginner | Prereq: 01 | ~4 min

Identity = visual translation of strategy. Strategy = soul, identity = face.

---

## 2.1 Design Process

```
STRATEGY → BRIEF → EXPLORATION(50 concepts) → REFINEMENT(3 finalists) → APPLICATION → GUIDELINES
```

> Never present logo in isolation. Show on website header, business card, billboard.

---

## 2.2 Logo Types

| Type | Description | Examples | Best For |
|------|------------|----------|----------|
| **Wordmark** | Company name as logo | Google, Coca-Cola | Distinctive names |
| **Lettermark** | Abbreviation | IBM, HBO | Long names |
| **Pictorial** | Icon/symbol | Apple, Twitter | Strong visual metaphor |
| **Abstract** | Geometric | Nike swoosh | Global, no language barrier |
| **Combination** | Symbol + wordmark | Adidas, Burger King | Versatility |
| **Emblem** | Text inside badge | Starbucks, Harley | Heritage, authority |

### Logo Family (not just one logo)
```
PRIMARY (full) → SECONDARY (stacked) → SYMBOL-ONLY (avatars) → WORDMARK-ONLY (narrow) → MONOCHROME
```

---

## 2.3 Brand Color System

| Role | Purpose | Quantity |
|------|---------|----------|
| Primary | Main identifier | 1-2 |
| Secondary | Supporting variety | 2-4 |
| Functional | UI (success, error, warning, info) | 4-6 |
| Neutrals | Backgrounds, text, borders | 5-8 shades |

### 60-30-10 Rule
```
60% neutral/background → 30% primary brand color → 10% accent (CTA, highlight)
```

---

## 2.4 Typography System

| Role | Font | Usage |
|------|------|-------|
| Display | High-impact, distinctive | Hero, billboards |
| Headings | Readable, characterful | H1-H3 |
| Body | Highly readable, neutral | Paragraphs, UI |
| Mono | Technical, precise | Code, data |

> Two-font max. One heading + one body. See `tipografi/03-font-pairing.md`.

## 2.5 Imagery Style

| Style | Feel | Brands |
|-------|------|--------|
| Editorial photography | Premium, magazine | Apple, Nike |
| Authentic / UGC | Real, unpolished | Airbnb |
| Illustration | Friendly, distinctive | Slack, Mailchimp |
| 3D / Abstract | Modern, tech | Stripe, Linear |
