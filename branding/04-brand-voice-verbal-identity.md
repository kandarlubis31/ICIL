# 🏷️ 04 — Brand Voice & Verbal Identity

> 🟡 Intermediate | Prereq: 01 | ~4 min

Verbal identity = how brand *sounds*. As consistent as logo. Voice = personality (fixed). Tone = mood (contextual).

---

## 4.1 Voice vs Tone

```
VOICE: "Friendly, confident, witty" (constant)
  ├── Celebratory → "You did it! 🎉"
  ├── Apologetic → "We messed up. Here's the fix."
  ├── Educational → "Let's break this down."
  └── Urgent → "Account needs attention. Act now."
```

---

## 4.2 Voice Dimensions (NNGroup)

```
FORMAL ←───────────────────→ CASUAL
SERIOUS ←───────────────────→ PLAYFUL
RESPECTFUL ←────────────────→ IRREVERENT
MATTER-OF-FACT ←────────────→ EMOTIONAL
```

### Voice Card Template
```
We ARE: ✅ Encouraging, ✅ Plain-spoken, ✅ Human
We ARE NOT: ❌ Corporate jargon, ❌ Condescending, ❌ Overly casual
Pillars: 1. Say it simply  2. Speak like a friend  3. Lead with empathy
```

---

## 4.3 Messaging Architecture

```
BRAND ESSENCE (1 sentence)
  → BRAND PROMISE (the "why")
    → VALUE PILLARS (3-5)
      → PROOF POINTS (features, data, testimonials)
```

> Every value pillar must have proof. Claims without evidence = fluff.

---

## 4.4 Naming Approaches

| Approach | Description | Examples |
|----------|------------|---------|
| Descriptive | Says what it does | PayPal, YouTube |
| Evocative | Suggests feeling | Nike, Amazon |
| Invented | Made-up, ownable | Kodak, Zillow |
| Metaphor | Symbolic | Apple, Sirius |

**Name test**: 100+ → filter → legal check(~5) → cultural check(~3) → audience test → decide

> ⚠️ Always check cultural meanings. "Nova" = "no va" (doesn't go) in Spanish.

---

## 4.5 Taglines — 5 Words Max

| Type | Example |
|------|---------|
| Descriptive | "The quickest way to send money" |
| Aspirational | "Think Different" |
| Action | "Just Do It" |
| Benefit | "Save Money. Live Better." |

---

## 4.6 Messaging × Journey

| Stage | Goal | Tone | Example |
|-------|------|------|---------|
| Awareness | "You have a problem. We exist." | Educational | "Tired of manual entry?" |
| Decision | "Choose us." | Reassuring | "Join 10K+ teams. Start free." |
| Onboarding | "You made the right choice." | Welcoming | "Let's get you set up." |
| Retention | "You're getting value." | Appreciative | "You saved 50h this month!" |
