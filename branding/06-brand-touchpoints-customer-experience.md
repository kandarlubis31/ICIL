# 🏷️ 06 — Brand Touchpoints & Customer Experience

> 🟡 Intermediate | Prereq: 02, 04 | ~3 min

Brand = sum of every customer interaction. Customers see ONE brand — not departments.

---

## 6.1 Touchpoint Ecosystem

```
PRE-PURCHASE          PURCHASE             POST-PURCHASE
Adverts, Social,       Website, Checkout,    Product, Support,
Reviews, Word-of-mouth  Sales, Onboarding    Emails, Loyalty

AWARENESS              CONVERSION            RETENTION
"Do I know you?"       "Should I buy?"       "Was it worth it?"
```

> If marketing promises "effortless" and support delivers "complicated" → brand is broken.

---

## 6.2 Touchpoint Audit Template

| Touchpoint | Stage | Owner | Score |
|------------|-------|-------|-------|
| Website | Awareness | Marketing | /10 |
| Checkout | Conversion | Engineering | /10 |
| Welcome email | Onboarding | CRM | /10 |
| Support chat | Post-purchase | Support | /10 |
| Packaging | Unboxing | Operations | /10 |
| Invoice | Post-purchase | Finance | /10 |

> **The gaps are the brand.** Fix touchpoints scoring 4-6/10 first.

---

## 6.3 The Brand Experience Curve

```
DELIGHT ↑
        │           ★ WOW MOMENT (Peak-End Rule)
EXPECTED ├─────────────────────────────
        │      ★ FRICTION POINT
PAIN    ↓
        ─────────────────────────────────→
        Awareness → Trial → Purchase → Use → Support → Advocacy
```

| Stage | Wow Opportunity | Example |
|-------|----------------|---------|
| Unboxing | Surprise & delight | Apple's slow-reveal packaging |
| Onboarding | Quick first win | Duolingo: lesson done in 60s |
| Milestone | Recognition | Spotify Wrapped |
| Support | Exceed expectations | Zappos upgrades shipping unprompted |

> Wow moments needed at **peak** and **end** only. Not everywhere.

---

## 6.4 Omnichannel Consistency Layers

| Layer | Must Match |
|-------|-----------|
| Visual | Logo, colors, typography, imagery |
| Verbal | Voice, tone, terminology, messaging |
| Behavioral | Response times, service quality |
| Experiential | Core value prop at every touchpoint |

| Breakdown | Fix |
|-----------|-----|
| Social voice ≠ website | Unified guidelines + review |
| App UI ≠ website | Shared design token system |
| Support = robotic | Voice training + templates |

---

## 6.5 Brand Health Metrics

| Metric | How to Track |
|--------|-------------|
| Awareness | Surveys, search volume, mentions |
| Sentiment | Social listening, reviews, NPS |
| Consistency | Touchpoint audit scores |
| Recall | Unaided recall surveys |
| Loyalty | Retention, repeat purchase rate |
| Advocacy | NPS, referrals, UGC |

```
NPS = % Promoters(9-10) - % Detractors(0-6)
50+ = Great | 0 = OK | -50 = Critical
```
