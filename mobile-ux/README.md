# 📱 Faculty of Mobile UX & Touch Design — Overview

> **7 Courses** | Level: Beginner → Advanced | Domain: Mobile User Experience & Touch Interaction Design

Mobile is not a smaller desktop. It's a fundamentally different interaction paradigm — driven by thumbs, gestures, context-switching, and constrained screen real estate. This faculty covers the complete spectrum of mobile UX design: from touch target physics and thumb zone ergonomics through navigation patterns, form optimization, haptics, and responsive/adaptive layout strategies. When an AI agent needs to design for phones, tablets, or any touch-first interface, this is the faculty to load.
