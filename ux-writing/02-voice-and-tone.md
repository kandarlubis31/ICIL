# ✍️ 02 — Voice & Tone

> 🟢 Beginner | Prereq: 01 | ~3 min

**Voice** = who you are (fixed personality). **Tone** = how you speak per situation (flexible).

---

## 2.1 NNGroup's 4 Tone Dimensions

```
FUNNY ←────────────────────→ SERIOUS
FORMAL ←────────────────────→ CASUAL
RESPECTFUL ←────────────────→ IRREVERENT
ENTHUSIASTIC ←──────────────→ MATTER-OF-FACT
```

| Brand | Position |
|-------|----------|
| Duolingo | Funny + Casual + Irreverent + Enthusiastic |
| Apple | Serious + Casual + Respectful + Matter-of-fact |
| Slack | Funny + Casual + Respectful + Enthusiastic |
| Stripe | Serious + Formal + Respectful + Matter-of-fact |

---

## 2.2 Tone Mapping by Situation

| Situation | Tone | Example |
|-----------|------|---------|
| Onboarding | Welcoming | "Welcome! Let's get started." |
| Success | Upbeat | "Woo! Payment successful." |
| Error | Empathetic, helpful | "Sorry, there's a problem. Try again?" |
| Warning | Serious, clear | "Account deleted in 3 days." |
| Loading | Reassuring | "Just a moment..." |
| Empty | Encouraging | "Nothing here yet. Add your first!" |
| Delete | Careful | "Delete permanently? Can't undo." |

---

## 2.3 Voice Chart Template

| Trait | DO | DON'T |
|-------|-----|-------|
| Helpful not bossy | "Try using filters." | "You must use filters." |
| Casual not sloppy | "Let's check it together." | "Oops error lol." |
| Smart not pretentious | "We use AES-256 encryption." | "Asymmetric cryptography per NIST standards." |

> For full brand-level verbal identity: see `branding/04-brand-voice-verbal-identity.md`.
