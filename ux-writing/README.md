# ✍️ Faculty of UX Writing — Overview

> **7 Courses** | Level: Beginner → Advanced | Domain: Content Design & Microcopy

This faculty teaches how words in UI can make products more usable, more human, and more engaging. From basic microcopy principles, voice & tone frameworks, to content design systems like Google and Shopify.
