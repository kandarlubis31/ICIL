# ✍️ 05 — Onboarding & Engagement Copy

> 🟡 Intermediate | Prereq: 01 | ~3 min

Onboarding = first impression. Engagement = what makes users return. Copy is the primary weapon for both.

---

## 5.1 Onboarding Goals & Rules

| Phase | Goal | Copy Rule |
|-------|------|-----------|
| 1. Value first | User understands benefit in <1 min | Benefit > Feature: "Save 5h/week" > "Cloud task manager" |
| 2. Action | User performs first key action (aha!) | 5-7 words per screen, 1 CTA |
| 3. Habit | User returns | 1 idea per screen, clear next step |

---

## 5.2 Touchpoint Copy Guide

| Touchpoint | Copy Goal | ❌ Bad | ✅ Good |
|-----------|-----------|--------|---------|
| Welcome | Short value prop | "Welcome to App" | "Manage finances 10× faster." |
| Permission | Benefit + choice | "[App] wants your location" | "Allow location for nearby restaurants? [Allow] [Later]" |
| Push Notification | Invitation, <30 chars | "You haven't opened the app..." | "Sarah mentioned you in Project" |
| Win-back Email | Remind value + 1 CTA | "We miss you!" | "We fixed [pain point]. Try free 30 days?" |
| Re-engagement | Show what they're missing | — | "You saved $150 this month with [Product]." |

---

## 5.3 Notification Types

| Type | Example |
|------|---------|
| Personal | "Sarah mentioned you in Project Phoenix" |
| Action | "Last day of your Premium trial!" |
| Value | "New insights from this week's data" |
| Social | "Your team just completed a milestone!" |

> ❌ Never guilt-trip: "You haven't opened the app in a while." → uninstall trigger.
