# ✨ Faculty of Animation & Motion — Overview

> **7 Courses** | Level: Beginner → Advanced | Domain: Motion Design & UI Animation

This faculty covers the entire spectrum of UI animation and motion design: from foundational principles (Disney's 12, when to animate), easing curves & timing, CSS implementation, micro-interactions, page transitions, scroll animations, to loading and skeleton screens. Designed so AI agents can create motion that feels natural, performant, and accessible.

### Beginner Path

