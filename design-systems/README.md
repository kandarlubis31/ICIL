# 🛠️ Faculty of Design Systems & Tools — Overview

> **7 Courses** | Level: Beginner → Advanced | Domain: Design Systems, Design Operations & Tooling

A design system is not a component library. It's the shared language between design and engineering — a living product that serves other products. This faculty covers the complete lifecycle: from strategic foundations and design token architecture through component library design, Figma workflows, governance & versioning, multi-brand theming, and ROI measurement. When an AI agent needs to build, maintain, or scale a design system, this is the faculty to load.
