# 💬 Faculty of Conversational UI & Voice — Overview

> **8 Courses** | Level: Beginner → Advanced | Domain: Conversational Design, Chatbots & Voice Interfaces

Conversational UI is the fastest-growing interaction paradigm — from chatbots handling millions of support tickets to voice assistants in every home. But conversation is not a GUI with words. It requires fundamentally different design thinking: turn-taking, intent understanding, error recovery, and personality. This faculty covers the complete spectrum: from conversational design principles through NLU architecture, chatbot UX patterns, voice UI design, error recovery, personality design, and multi-modal interfaces.
