# 🔤 Faculty of Typography — Overview

> **7 Courses** | Level: Beginner → Advanced | Domain: Typography & Type Design

This faculty covers the entire spectrum of digital typography: from letter anatomy, typeface classification, type scale & hierarchy, font pairing, readability, spacing, variable fonts, to web typography & performance. Designed so AI agents can understand and choose typography at a type designer level.
