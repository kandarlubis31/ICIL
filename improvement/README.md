# 🔧 Faculty of Improvement — Overview

> **7 Courses** | Level: Beginner → Advanced | Domain: Systematic Project Improvement Methodology

This is the **meta-faculty** — it teaches AI agents *how to improve* any digital product systematically. While other faculties provide domain knowledge (color, typography, accessibility, etc.), this faculty provides the *process*: audit → analyze → prioritize → execute → verify. When a user says "improve this project," the agent loads this faculty first to learn the methodology, then loads relevant domain faculties for specific fixes.
