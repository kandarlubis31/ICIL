# 🧩 Faculty of Design Patterns — Overview

> **8 Courses** | Level: Intermediate → Advanced | Domain: UI Patterns & Interaction Design

This faculty is a **complete UI patterns catalog** — from container patterns (modal, drawer, popover), form design, navigation, search, data display, feedback states, selection inputs, to complex components. This is the "encyclopedia" that enables AI agents to choose the right pattern for every situation.
