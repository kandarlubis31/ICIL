# 🧠 Faculty of Human Cognition — Overview

> **7 Courses** | Level: Beginner → Advanced | Domain: Cognitive Science & The Science of Human Thinking

This is the **foundational science faculty** — it teaches how the human brain actually works. While `ux-psikologi` covers psychology *applied to design*, this faculty covers the **fundamental cognitive science** behind all human thinking: information processing, dual process theory, perception, memory, decision-making, problem-solving, and the interplay of language, emotion, and metacognition. When an AI agent needs to understand *why* humans think the way they do, this is the faculty to load.

## 🎯 Learning Paths

### Beginner Path (Cognitive Foundations)
```
01 → 02 → 03
```
Understand the brain's information processing architecture, the two-system model of thinking, and how perception and attention filter reality.

### Intermediate Path (Memory, Decisions & Creativity)
```
01 → 02 → 03 → 04 → 05 → 06
```
Add memory systems (how humans store and forget), decision-making heuristics (how humans choose), and problem-solving & creativity (how humans innovate).

### Advanced Path (Full Cognition Science)
```
01 → 02 → 03 → 04 → 05 → 06 → 07
```
Complete the full curriculum: from architecture to language, emotion, metacognition, learning theories, expertise development, and neuroplasticity.
