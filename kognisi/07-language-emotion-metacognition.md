# 🧠 07 — Language, Emotion & Metacognition

> **Level: 🔴 Advanced** | Prerequisite: 01, 02 | Est. reading: 8 min

The three layers of cognition that shape every interaction: words, feelings, and self-awareness. Master these and your UI becomes a conversation, not a struggle.

---

## 7.1 Language Processing in UI

| Principle | Application |
|-----------|------------|
| **Lexical access** — word recognition | Use high-frequency words; avoid jargon |
| **Semantic priming** — related words speed processing | Consistent terminology across screens |
| **Pragmatics** — context shapes meaning | "Delete" means different things in email vs file manager |
| **Grice's Maxims** — cooperative communication | Be relevant, clear, concise (see ux-writing faculty) |

---

## 7.2 The Sapir-Whorf in UI

> The language your UI uses shapes how users think about your product.

| UI Language | User Perception |
|-------------|----------------|
| "Create project" | Productive tool |
| "New thing" | Casual, playful |
| "Initialize workspace" | Technical, professional |
| "Get started!" | Friendly, guided |

## 7.3 Emotion & Cognition

| Emotion | Effect on Cognition | UI Strategy |
|---------|-------------------|-------------|
| **Frustration** | Tunnel vision, less creative | Clear error recovery; never blame user |
| **Anxiety** | Impaired working memory | Reduce time pressure; show progress |
| **Joy** | Broadened attention, creative | Delightful micro-interactions; surprise rewards |
| **Trust** | Reduced cognitive monitoring | Consistency, transparency, social proof |
| **Flow** | Optimal performance + engagement | Clear goals, immediate feedback, balanced challenge |

---

## 7.4 Emotional Design (Don Norman's 3 Levels)

| Level | Processing | Design Target |
|-------|-----------|--------------|
| **Visceral** | Automatic, pre-conscious | Beautiful aesthetics — first impression |
| **Behavioral** | Use-driven, functional | Usability, performance — the experience |
| **Reflective** | Conscious, meaning-making | Brand story, identity, values — the memory |

```
Visceral: "Wow, this app is beautiful"
Behavioral: "This is so easy to use"
Reflective: "I love what this brand stands for"
```

---

## 7.5 Metacognition — Thinking About Thinking

| Metacognitive State | UX Impact |
|--------------------|-----------|
| **Feeling of Knowing (FOK)** — "I know this" | Users skip help — but may be wrong. Provide subtle guidance. |
| **Tip-of-Tongue (TOT)** — "I almost remember" | Search autocomplete; related suggestions |
| **Judgment of Learning (JOL)** — "I'll remember this" | Users overestimate retention. Spaced onboarding beats one tutorial. |
| **Confidence** — over vs under | Calibrate: confirm critical actions, don't question trivial ones |

---

## 7.6 Designing for Emotional States

| User arriving with... | Design Response |
|----------------------|----------------|
| **Anger** (bug, lost data) | Apologize, recover data, don't deflect blame |
| **Confusion** (complex task) | Explain, scaffold, don't overwhelm |
| **Excitement** (new feature) | Onboard quickly, don't interrupt flow |
| **Indifference** (routine task) | Be invisible; get out of the way |
