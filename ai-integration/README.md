# 🤖 Faculty of AI Integration & LLM Engineering — Overview

> **10 Courses** | Beginner → Advanced | Domain: LLM Engineering, RAG & AI Tool Building

Covers the engineering side of AI: prompt engineering, RAG architecture, embeddings, LLM orchestration, evaluation, safety, and MCP tool integration. Distinct from `conversational-ui/` (which covers chatbot UX, NLU design, and voice interfaces).
