================================================================
📌 WHAT IS THIS?
================================================================

ICIL (Intelligence Campus Interactive Library) is a SUPER COMPREHENSIVE KNOWLEDGE BASE
purpose-built as a REFERENCE for AI AGENTS.

Think of it like this: An AI agent is like a student. It can code,
design, analyze — but its knowledge is limited to its training data.

This campus is a SACRED LIBRARY that AI agents can load as
"context injection." Once loaded, the agent instantly gains
EXPERT-LEVEL KNOWLEDGE in specific domains.

================================================================
🎯 THE BIG VISION
================================================================

"Build AI agents that can design, write, and code LIKE HUMANS —
with expert-level quality standards."

User just says:
  "Bro, improve this website's UI to feel more human."
  "Build a landing page that feels luxurious and elegant."
  "Write engaging microcopy for the onboarding flow."
  "Design ethical AI systems that respect user privacy."

→ AI agent AUTO-DETECTS which faculties are relevant.
→ Loads the context.
→ Executes with EXPERT-LEVEL quality.

No need for the user to pick faculties — the AI figures it out itself.

🧠 HOW IT WORKS (FOR AI AGENTS)
================================================================

STEP 1 — Agent receives a user prompt:
  "Make a dashboard UI that's more engaging so users don't bounce."

STEP 2 — Agent reads index.json → matches keywords:
  "engaging"      → ux-psikologi (persuasion, gamification)
  "don't bounce"  → ux-psikologi (cognitive biases, choice paralysis)
  "dashboard UI"  → warna (palette, accessibility)
                  → ux-writing (microcopy)

STEP 3 — Agent loads relevant context:
  [load ux-psikologi/02-cognitive-biases.md]
  [load ux-psikologi/05-persuasion-behavioral-design.md]
  [load warna/06-color-accessibility.md]
  [load ux-writing/03-component-microcopy.md]

STEP 4 — Agent EXECUTES with expert-level knowledge.

OR use the auto-router CLI:
  node load-context.js "build an accessible landing page"
  node load-context.js "prompt" --json    # programmatic
  node load-context.js --list             # all faculties

OR use the MCP server for AI agent integration:
  { mcpServers: { icil: { command: "node", args: ["mcp-server.js"] } } }
  10 tools: search_campus · load_course · list_faculties · search_across
            · compare_courses · get_campus_stats · search_by_faculty
            · get_course_prerequisites · export_course · recommend_learning_path

🚀 HOW TO USE THIS PROJECT
================================================================

### Quick Start (for AI agents)
```bash
# Install dependencies
npm install

# Auto-route a prompt
node load-context.js "design an accessible form for mobile"

# List all faculties
node load-context.js --list

# Interactive mode
node load-context.js --interactive

# Programmatic usage
const { searchCampus } = require('./campus-core');
const result = searchCampus("your prompt here");
```

### MCP Server (Claude Desktop / Cursor)
```json
{
  "mcpServers": {
    "icil": {
      "command": "node",
      "args": ["path/to/inteligence_mas-aul/mcp-server.js"]
    }
  }
}
```

### Validate & Test
```bash
npm run ci:validate     # Schema + duplicates + cross-refs
npm run ci:gate         # Full CI with eval gate
npm run eval            # Run eval suite
```
