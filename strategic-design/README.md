# 🧭 Faculty of Strategic Design & Product Thinking — Overview

> **7 Courses** | Level: Beginner → Advanced | Domain: Strategic Product Design & Decision-Making

This is the **strategic thinking faculty** — it teaches AI agents how to think like product leaders: frame problems, discover what to build, set strategic direction, make decisions under uncertainty, and bridge strategy to execution. While `improvement/` provides the tactical *methodology* for systematic improvement and `kognisi/` explains the *science* of human cognition, this faculty fills the missing layer: **how to think strategically** — the planning brain behind every great product decision.

## 🎯 Learning Paths

### Beginner Path (Design Thinking Core)
```
01 → 02
```
Learn the Double Diamond process and strategic problem framing — the foundation of human-centered product thinking.

### Intermediate Path (Product Strategy)
```
01 → 02 → 03 → 04
```
Add product strategy, outcome thinking, and roadmapping — the complete strategic planning toolkit.

### Advanced Path (Full Strategic Mind)
```
01 → 02 → 03 → 04 → 05 → 06 → 07
```
Complete strategic education: design process → problem framing → product strategy → roadmapping → first principles reasoning → decision-making under uncertainty → execution mastery.
